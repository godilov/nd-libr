# NdLibr Library

**NdLibr meta-library**

## Start

```toml
[dependencies]
ndlibr = "*"
```

**Feature-flags:**

- `all` - enables all features.
- `assert` - enables `ndassert` dependency.
- `fwd` - enables `ndfwd` dependency.
- `ops` - enables `ndops` dependency.
- `crypto` - enables `ndcrypto` dependency (exposed via `ndlibr::crypto`).
- `ext` - enables `ndext` dependency (exposed via `ndlibr::ext`).
- `mem` - enables `ndmem` dependency (exposed via `ndlibr::mem`).
- `num` - enables `ndnum` dependency (exposed via `ndlibr::num`).
