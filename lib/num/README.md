# NdNumbers Library

## Start

```toml
[dependencies]
ndnum = "*"
```

**Feature-flags:**

- `all` - enable all features.
- `std` - enable `stdlib`.
- `dyn` - enable dyn-allocated long numbers.
- `simd` - enable SIMD-optimized long numbers.
- `rand` - enable `rand` dependency.
- `rayon` - enable `rayon` dependency.
- `const-time` - enable const-time operations.
